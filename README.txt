#Calculadora Credilight Mejoravit

Es una app web desarrollada con Python y Streamlit que permite calcular descuentos y generar un PDF profesional para créditos de clientes, se diseño para facilitar el trabajo en oficina y automatizar los cálculos de forma rápida y confiable.

#Caracteristicas

-Interfaaz moderna y simple con logoy diseño corporativo
-Calcula descuentos y hace lógica del negocio
-Genera PDF profesional

#Tecnologias utilizadas
-Python 3.x
-Streamlit
-ReportLab
